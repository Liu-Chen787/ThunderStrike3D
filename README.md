# Thunder Strike 3D

A 3D arcade-style space shooter developed in Unity.

## Overview
Thunder Strike 3D is inspired by classic vertical scrolling shooters.
The player controls a fighter aircraft and must eliminate enemy units
to gain score and win the game.

## Core Features
- 3D player movement (X-Y plane, fixed Z)
- Enemy spawning system (limited + elite enemies)
- Shooting and collision detection
- Explosion VFX with sound
- Score system and win condition
- Player health system
- Speed Boost skill with cooldown
- Item drop system (Heart & Lightning)
- Responsive UI (HealthBar, SkillCD, ItemBar)

## Technology
- Unity 6 (6000.3 LTS)
- C#
- Unity Input System
- URP
